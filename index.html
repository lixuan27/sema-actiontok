Read the codes out as a sequence with a stop symbol, chosen step by step:
for t = 1 … H:  score vocabulary + stop
  t = 1 → forbid stop  (intent must lead)
  t = H → force stop  (the code always closes)
  sample zt ;  break when stop is chosen
Why the two rules: a guaranteed head token gives the shared prefix meaning; a guaranteed close keeps every code serialised. Length between them is learned.

Output: z = [ intent, detail, … , stop ] — the model sets the length per sample.

Decoding: any prefix goes through one shared decoder, a legal projection, and a per-body adapter.
